# GAV

GAV — это социальная платформа для владельцев собак.
Проект объединяет в одном приложении общение, заботу о питомцах и поиск компании для прогулок: пользователи могут знакомиться с другими собаководами, создавать посты, вести трекер прививок и получать полезные рекомендации от профессионалов.

## Идея проекта

Главная цель GAV — помочь владельцам собак собрать удобное и безопасное сообщество вокруг повседневных прогулок, общения и ухода за питомцами.
Приложение рассчитано на людей, которые хотят не только делиться моментами из жизни со своей собакой, но и находить новых знакомых поблизости, следить за важной информацией о здоровье питомца и получать практические советы.

## Основные возможности

- профили пользователей с фото, био, списком собак, подписчиками и постами
- карточки собак и отображение собак в формате сторис
- лента постов с социальным взаимодействием
- поиск компаньонов для прогулок и других владельцев собак поблизости
- карта с отметками гуляющих пользователей и собак
- трекер прививок для своих собак
- мессенджер для общения между пользователями
- рекомендации и полезный контент от специалистов

## Концепция продукта

GAV — это не просто социальная сеть про собак.
Это единое цифровое пространство, где можно:

- искать людей для совместных прогулок
- формировать локальное dog-friendly сообщество
- хранить важную информацию о питомце в одном месте
- делиться историями, опытом и полезными советами

## Структура репозитория

- `GavApp/` — мобильное приложение
- `backend/` — backend-часть и серверная логика
- `api/` — API-контракты и связанные артефакты

## Предполагаемый сценарий использования

1. Пользователь создаёт профиль и добавляет одну или несколько собак.
2. Собаки отображаются в профиле в формате сторис.
3. Пользователь публикует посты, просматривает ленту и взаимодействует с другими владельцами собак.
4. На карте можно найти людей, гуляющих поблизости, и увидеть отметки их собак.
5. Владелец ведёт трекер прививок и хранит важную информацию о питомце.
6. Пользователи общаются в мессенджере и получают полезные рекомендации.

## Зачем нужен этот проект

Сейчас владельцам собак часто приходится пользоваться сразу несколькими сервисами: социальными сетями для общения, заметками для хранения информации, картами для поиска встреч и мессенджерами для координации.
GAV объединяет эти сценарии в одной экосистеме, специально ориентированной на собак и их владельцев.

## Текущий статус

Проект находится в активной разработке.
На текущем этапе в репозитории уже есть мобильная и backend-часть, а сама идея постепенно оформляется в полноценную социальную платформу для владельцев собак.
